# VXL
VXL to prosta gra voxelowa napisana w Javie przy użyciu LWJGL 3.

## Ważne!
Gra jest w wyjątkowo wczesnej fazie rozwoju, większość planowanych rzeczy nie jest jeszcze zaimplementowana.
